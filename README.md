# Habit Tracker
A simple Habit Tracker built using HTML, CSS, and JavaScript.

## Features
* Add new habits
* Press Enter to add habits
* Mark habits as completed
* Strike-through completed habits
* Double-click to delete habits
* Save habits using Local Storage
* Automatically load habits after page refresh
* Responsive design

## Technologies Used
* HTML
* CSS
* JavaScript
* Local Storage

## How to Use
1. Enter a habit in the input field.
2. Click **Add** or press **Enter**.
3. Tick the checkbox to mark a habit as completed.
4. Double-click a habit to delete it.
5. Habits are automatically saved in the browser.
